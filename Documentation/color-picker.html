Bootstrap Colorpicker

Fancy and customizable colorpicker plugin for Twitter Bootstrap.

Official website and detailed documentation: https://farbelous.github.io/bootstrap-colorpicker/

Basic example

  • HTML
  • Javascript
<div class="input-group">
    <span class="input-group-addon"><i class="zmdi zmdi-invert-colors"></i></span>

    <div class="form-group color-picker">
        <input type="text" class="form-control color-picker__value" value="#03A9F4">
        <i class="color-picker__preview" style="background-color: #03A9F4"></i>

        <i class="form-group__bar"></i>
    </div>
</div>
if ($('.color-picker')[0]) {
    $('.color-picker__value').colorpicker({
        horizontal: horizontal
    });

    $('body').on('change', '.color-picker__value', function () {
        $(this).closest('.color-picker').find('.color-picker__preview').css('backgroundColor', $(this).val());
    });
}

Requirements:

  • /scss/inc/vendor-overrides/_bootstrap-colorpicker.scss
  • /vendors/bower_components/bootstrap-colorpicker/dist/css/bootstrap-colorpicker.css
  • /vendors/bower_components/bootstrap-colorpicker/dist/js/bootstrap-colorpicker.min.js
